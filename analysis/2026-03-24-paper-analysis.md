## 论文1: FASTER - Rethinking Real-Time Flow VLAs

**来源**: arXiv 2603.19199 | 项目页: https://innovator-zero.github.io/FASTER

### 摘要
针对VLA模型的实时执行问题，提出Fast Action Sampling for ImmediaTE Reaction (FASTER)。通过引入Horizon-Aware Schedule，将近期动作去噪压缩到单步，实现消费级GPU上的毫秒级反应延迟。在乒乓球任务中验证了前所未有的实时响应能力。

### 方法动机
- **痛点**: Flow-based VLA需要完成所有采样步骤才能输出动作，导致反应延迟
- **关键洞察**: 反应时间由TTFA (Time to First Action) 和执行视界共同决定，呈均匀分布
- **问题定位**: 标准常量调度迫使系统完成所有采样步骤才能开始运动，形成延迟瓶颈

### 核心方法

#### 1. 反应时间理论分析
```
反应时间 ~ Uniform(TTFA, TTFA + Execution_Horizon)
```
- TTFA：首次动作输出时间
- 执行视界：动作块的时间跨度

#### 2. Horizon-Aware Schedule
- **自适应优先级**: 近期动作优先采样
- **单步压缩**: 将近期动作去噪压缩至1步（如π0.5和X-VLA）
- **长视界保留**: 保持长时程轨迹质量

#### 3. 流式客户端-服务器管道
- 降低有效反应延迟
- 适用于消费级GPU部署

### 关键结果
| 指标 | 结果 |
|------|------|
| 近期动作去噪 | 10倍压缩（10步→1步） |
| 乒乓球任务 | 实时响应验证 |
| 部署平台 | 消费级GPU可用 |

### 创新点
1. **重新定义"反应"**: 从动作块视角系统分析反应延迟
2. **均匀分布理论**: 首次揭示反应时间的概率特性
3. **自适应调度**: 突破常量调度的固有瓶颈

### 与Charles研究方向的关联

#### CT介入机器人 (⭐⭐⭐⭐⭐)
- **实时穿刺控制**: 毫秒级延迟满足医疗场景需求
- **关键阶段优化**: 穿刺阶段可全步采样，导航阶段可压缩

## 论文2: StructVLA - World Models as Structured Planners

**来源**: arXiv 2603.12553

### 摘要
将生成式世界模型重构为显式结构化规划器，预测稀疏、物理意义明确的结构化帧（基于运动学拐点），而非稠密未来帧。在LIBERO上达到94.8%成功率，SimplerEnv-WidowX上75%。

### 方法动机
- **痛点**: 稠密未来预测引入视觉冗余和累积误差，导致长时程规划漂移
- **现有方案不足**: 稀疏方法使用高层语义子任务或隐式潜在状态，缺乏显式运动学基础
- **洞察**: 运动学拐点（gripper transitions, kinematic turning points）是任务进度的物理里程碑

### 核心方法

#### 1. 结构化帧定义
```
结构化帧 = 运动学拐点时刻的视觉状态
├── Gripper transitions (开/闭转换)
├── Kinematic turning points (速度方向变化)
└── 任务进度里程碑
```

#### 2. 两阶段训练范式
```
Stage 1: 世界模型训练
├── 输入: 当前状态 + 动作
└── 输出: 结构化帧预测

Stage 2: 动作映射优化
├── 输入: 结构化预视
└── 输出: 低层动作序列
```

#### 3. 统一离散Token词汇表
- 视觉和动作共享词汇表
- 便于跨模态对齐

### 关键结果
| 基准 | 成功率 |
|------|--------|
| LIBERO | 94.8% |
| SimplerEnv-WidowX | 75.0% |
| 真实机器人 | 强泛化能力 |

### 创新点
1. **稀疏物理帧**: 替代稠密视觉预测
2. **运动学基础**: 显式物理意义，非隐式潜在
3. **规划-执行桥梁**: 结构化帧直接指导低层控制

### 与Charles研究方向的关联

#### Brain-Cerebellum架构 (⭐⭐⭐⭐⭐)
- **Brain层**: 结构化规划器（运动学里程碑）
- **Cerebellum层**: 低层动作执行
- **完美对应**: 提供了可落地的分层框架

#### 数字孪生+VLA (⭐⭐⭐⭐)
- **结构化里程碑**: 仿真环境中可精确标注
- **虚实一致性**: 结构化帧比稠密帧更易对齐
- **Sim2Real**: 运动学拐点是任务无关的通用特征

#### CT介入机器人
- **穿刺里程碑**: 进针、到达目标、撤针
- **安全监控**: 拐点时刻验证位置正确性

## 论文3: VLA-Thinker - Thinking-with-Image Reasoning

**来源**: arXiv 2603.14523 | 项目页: https://cywang735.github.io/VLA-Thinker/

### 摘要
首个支持"思考图像"推理的VLA模型，将视觉感知建模为动态可调用的推理动作。两阶段训练：SFT冷启动 + GRPO强化学习对齐。LIBERO成功率97.5%。

### 方法动机
- **痛点**: 现有VLA使用文本CoT，视觉输入作为静态上下文
- **限制**: 无法主动重新观察环境，无法解决长时程任务中的歧义
- **洞察**: 感知应该是动态可调用的推理动作

### 核心方法

#### 1. 思考图像框架
```
VLA-Thinker
├── 文本推理动作 (Text Reasoning)
├── 图像思考动作 (Image Thinking)
│   ├── 主动重新观察 (Active Re-observation)
│   └── 歧义解决 (Ambiguity Resolution)
└── 物理执行动作 (Physical Action)
```

#### 2. 两阶段训练
```
Stage 1: SFT冷启动
├── 视觉CoT数据集
└── 激活结构化推理 + 工具使用行为

Stage 2: GRPO强化学习
├── 奖励信号: 任务成功率
└── 对齐完整推理-动作轨迹
```

#### 3. 多模态具身CoT
- 文本 + 图像混合推理
- 动态感知调用机制

### 关键结果
| 基准 | 成功率 |
|------|--------|
| LIBERO | 97.5% |
| RoboTwin 2.0 | 强增益 |

### 创新点
1. **感知即动作**: 视觉感知建模为可调用动作
2. **动态观察**: 任务执行中主动获取新信息
3. **GRPO对齐**: 强化学习优化完整轨迹

### 与Charles研究方向的关联

#### 可解释VLA决策框架 (⭐⭐⭐⭐⭐)
- **推理可视化**: 思考图像可追踪决策过程
- **Failure Mode诊断**: 可定位是感知错误还是动作错误
- **人类验证**: 中间图像思考可被人类检查

#### CT介入机器人
- **动态观察**: 穿刺前主动获取CT/超声确认
- **风险验证**: 高风险操作前重新观察
- **安全决策**: 图像思考作为安全检查点

#### Brain-Cerebellum架构
- **Brain层**: 图像思考 + 文本推理

- **动态调度**: 感知动作与物理动作分离

## 论文4: Robotic Ultrasound Makes CBCT Alive

**来源**: arXiv 2603.10220 | 代码: https://github.com/anonymous-codebase/us-cbct-demo

### 摘要
利用机器人超声作为动态代理，推断组织运动并实时更新静态CBCT切片。提出USCorUNet学习变形感知相关性表示，实现无重复辐射暴露的动态引导。

### 方法动机
- **痛点**: CBCT是静态的，无法监控呼吸、探头压力、手术操作引起的软组织变形
- **后果**: 导航出现偏差，影响介入精度
- **洞察**: 超声是动态的，可作为变形代理

### 核心方法

#### 1. 多模态校准
```
校准流程:
├── 初始对齐: 标定参数
├── 刚性精化: LC2-based rigid refinement
└── 结果: 准确的多模态对应关系
```

#### 2. USCorUNet架构
```
USCorUNet
├── 输入: 超声流
├── 监督: 光流引导监督
├── 输出: 变形感知相关性表示
└── 特点: 轻量级、实时
```

#### 3. 变形场估计与传递
```
超声 → USCorUNet → 相关性表示 → 密集变形场 → CBCT更新
                                          ↓
                                    空间正则化
```

### 关键结果
- **实时端到端CBCT切片更新**
- **物理合理的变形估计**
- **无重复辐射暴露**

### 创新点
1. **超声代理**: 用超声推断CBCT变形
2. **光流监督**: 无需配对的变形标注
3. **零辐射更新**: 安全的动态引导

- **动态更新**: 解决CBCT静态限制
- **临床价值**: 提升穿刺精度和安全性

#### 技术融合方案
```
CT介入VLA + CBCT超声融合:

├── CBCT静态参考: 3D解剖上下文
```
