## 💡 关键洞察

### 1. CT领域正在复制VLA成功路径

```
CV领域: ImageNet → ResNet → CLIP → VLM
                  ↓
CT领域: CT扫描 → Merlin → CT-VLA → 手术机器人
                            ↑
                    我们在这里!
```

**启示**: 
- Merlin填补了CT感知的基础设施空白
- 下一步是CT-VLA (加入动作输出)
- **这是我们的机会窗口**

### 2. 医疗VLA是新兴增长点

根据VLA综述统计:
- 人形机器人: 27%
- 自动驾驶: 23%
- 工业机器人: 18%
- **医疗机器人: 12%** ← 增长最快
- 精准农业: 10%
- AR导航: 10%

**启示**: 
- 医疗VLA竞争尚未激烈
- ICLR 2026无竞品
- ICRA 2026 VLA Workshop提供评估平台

### 3. Gaze-VLA开启共享控制新范式

**传统共享控制**:
```
医生操作 → 机器人执行 (100% 医生意图)
```

**Gaze-VLA共享控制**:
```
医生注视 → VLA注意力聚焦 → 风险加权动作
          (人类意图 + 机器理解)
```

**优势**:
- 无需显式指令，意图自然表达
- 可解释性强，注意力可视化
- 无推理开销，实时安全
