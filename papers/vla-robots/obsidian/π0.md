# π0: A Vision-Language-Action Flow Model for General Robot Control

## 核心贡献
- Flow Matching 用于机器人动作生成
- 视觉-语言-动作端到端模型
- Physical Intelligence 公司发布
- 基于预训练视觉编码器 + 语言模型

## 关键创新
- Flow Matching 替代传统扩散模型
- 统一不同机器人形态
- 泛化能力强

## 与其他VLA的关系
- RT-2 → OpenVLA → π0 发展脉络
- π0 是当前最强 VLA 之一
- 启发了后续 RobustVLA 等工作
